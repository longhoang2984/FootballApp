### Requirements
 Tool | Version 
--- | ---
XCode | 14.0.0
iOS | 16.0
Device | 14 Pro

#### Features:
-   Display previous and upcoming matches.
-   Select a team for team details.
-   Filter matches by team(s).
-   Watch highlights of a previous match. (Simple with AVPlayerController)

3. Switch to `FootballApp` and run `CMD+R`
